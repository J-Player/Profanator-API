# Profanator-API

É uma API reativa desenvolvida em **Spring WebFlux** com o propósito de fornecer dados relacionados aos itens do **Profane**.

Menu:
- [Rotas](https://github.com/J-Player/Profanator-API#rotas)
- [Dados](https://github.com/J-Player/Profanator-API#dados)
- [Dependências](https://github.com/J-Player/Profanator-API#depend%C3%AAncias)

# Rotas

Para informações detalhadas sobre as rotas, acesse o _Swagger-UI_ da API enquanto o servidor estiver *rodando*.

Sintaxe do URI do Swagger: http://&lt;host&gt;:&lt;port&gt;/swagger-ui.html
  
  Exemplo: http://localhost:8080/swagger-ui.html.

# Dados

Como o Profane ainda não tem uma API para desenvolvedores, foi necessário criar todos os dados (itens) da API manualmente, um por um.

Atualmente há um total de **184 itens** cadastrados em um banco de dados provisório.

Os dados **NÃO estão incluídos no projeto** pois ainda não decidi que tipo de banco de dados irei utilizar para produção.

# Dependências

- Reactor Netty
- Spring Boot Starter Data R2DBC
- Spring Boot Starter Security
- Spring Boot Starter Validation
- Spring Boot Starter Webflux

Dependências opcionais:
  - Blockhound
  - Lombok
  - Spring Boot Devtools
  - Springdoc OpenAPI Webflux UI
